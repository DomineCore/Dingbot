# Dingbot : 一个轻量级的SDK，专为钉钉群机器人打造
## 介绍
　　这是一个轻量级的SDK，专为钉钉群机器人打造。可在所有python环境下运行。  
　　本SDK依靠[钉钉支持文档](https://ding-doc.dingtalk.com/doc#/serverapi2/qf2nxq)而编写，支持所有机器人的操作。
