# Dingbot : 一个轻量级的SDK，为钉钉群机器人打造
## 介绍
这是一个轻量级的SDK，依靠[钉钉支持文档](https://ding-doc.dingtalk.com/doc#/serverapi2/qf2nxq)而编写，支持所有机器人的操作。  
## 特点
- [x] 超小的代码量 -- 仅有不到200行  
- [x] 支持所有官方提供的钉钉群机器人操作  
- [x] 对接口提供了标准封装  
- [ ] ~支持命令行操作~  
- [x] 大量的格式支持，可简易地分享网页、图片
